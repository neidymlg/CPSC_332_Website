About

This is a simple web application that allows users to query a database of students, professors, courses, and grades. The database is a mock database and the queries are simple. The queries are as follows:

  • For the professors: Given the social security number of a professor, list the titles, classrooms, meeting days and time of his/her classes.
  • For the students: Given a student's CWID, list the classes this student takes with their grades.
  • For the courses: Given a course number, list the sections of the course, including the classrooms, the meeting days and time, and the number of students enrolled in each section.
  • For the grades: Given a student's CWID, list the courses this student takes with their grades.
